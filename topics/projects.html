Projects

I have been involved with the following projects either as the main researcher, supervisory team or developer.

Draw my Graph (DmG)

Draw my Graph is a Network/Graph visualization tool. It is the result of my PhD research topic (Novel heuristic approaches to iterative graph drawing) This project contains a few novel graph drawing techniques. The tool can be downloaded from TBC

I have been involved as the main researcher and tool developer for this project.

Large network visual analytics tool (NetvizGL)

NetvizGL is a graph visualization based on multilevel graph drawing algorithms. This is an FYP work by William O'Dwyer. A demo of the tool can be found at: http://bdarg.org/projects The github page can be found at: https://github.com/Willieodwyer/netvizGL

I have been involved as a member of supervisory team for this project.

Feature Location INTerface Specification (FLINTS)

FLINTS is a semi-automated, near-complete feature location tool. It embeds multiple code search techniques such as Textual search technique, query recommendation, invocation analysis. A demo of the tool can be found at: https://arc.lero.ie/flints/

I have been involved as a researcher and tool developer for this project.

Source Code Analyser and Inspector Tool (SCAIT)

SCAIT is a source-code analyser tool for Java that perform low-level analysis such as artefact dependencies, invocation analysis and etc. SCAIT is designed very flexible so any new analysis may be embedded with minimum effort.
